# UFW Firewall Configuration

## Objective
To configure a basic firewall using UFW on Linux.

## Steps
1. Installed UFW
2. Allowed SSH (port 22)
3. Denied HTTP (port 80)
4. Enabled firewall
5. Checked firewall status

## Result
Firewall is active with correct rules.
